## 起動方法

- **Tools → Aramaa → おちびちゃんズ化ツール（Ochibi-chans Converter Tool）**
- **GameObject → Aramaa → おちびちゃんズ化ツール（Ochibi-chans Converter Tool）**

## ツールウィンドウの読み方（上から順）

### ① 変換元アバター（Hierarchyで選択）

- **Scene上**のアバターを、Hierarchyで 1 つ選択します。
- Project上の Prefab を選んでも対象になりません。

> ダイアログ「Hierarchy で元のアバターを 1 つ選んでください。」が出る場合は、この選択ができていません。
{: .warning }

### ② 変換先おちびちゃんズ（プルダウン）

- 顔メッシュ（Viseme用メッシュ）を手がかりに、`Assets/夕時茶屋` 配下から **候補を自動検出**して一覧表示します。
- 候補が出ない場合は、次の「手動指定」で進められます。

> 候補が出ないときは [トラブルシューティング]({{ "/troubleshooting/" | relative_url }}) の「候補が出ない」を見てください。
{: .tip }

### おちびちゃんズ Prefab（手動指定）

- Project から **おちびちゃんズ Prefab をドラッグ＆ドロップ**します。
- 自動検出の候補が出る環境でも、手動指定してOKです。

### オプション: MA Bone Proxy のずれ対策

「**MA Bone Proxyで設定している髪・小物がずれる場合に合わせる**」を ON にすると、複製後に MA Bone Proxy の処理を行い、ずれを軽減します（ツール内ヘルプ: 「ONにすると、複製後にMA Bone Proxyの処理を行い、ずれを軽減します。」）。

> OFFのままで問題ないアバターも多いので、「ずれるときだけON」が基本です。
{: .note }

### オプション: ログを表示する

「**ログを表示する**」を ON にすると、処理後にログウィンドウが開きます。

- どの処理が実行されたか（スキップされたか）
- エラーや例外

をまとめて確認できます。困ったときの Discord 相談にも、そのまま貼れて便利です。

## 実行ボタン（コピー→変換）

「**③ 実行（コピー→変換）**」を押すと、次の順序で処理します。

1. **Ctrl+D相当で複製**（元アバターは変更しません）
2. （任意）MA Bone Proxy の処理
3. 複製物の **Blueprint ID をクリア**（元IDの引き継ぎ防止）
4. 変換先おちびちゃんズの設定を複製物に同期

> 変換後に「アップロードすると新規扱いになる」挙動は、Blueprint ID をクリアする仕様のため正常です。
{: .tip }

## よくある操作パターン

### パターンA: まずは1回だけ変換したい

1. 変換元アバターを選択
2. 変換先を自動検出（or 手動指定）
3. 実行
4. 見た目とConsoleを確認

### パターンB: ずれやすい髪・小物がある

1. 上と同じ
2. **MA Bone Proxyのずれ対策をON**
3. 実行
4. まだズレる場合は [困ったとき]({{ "/troubleshooting/" | relative_url }}) へ
